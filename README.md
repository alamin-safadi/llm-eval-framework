# LLM Evaluation Framework (Bias Detection)

## Overview
This project demonstrates LLM-as-a-Judge and bias detection.

## Features
- Candidate LLM generates answers
- Judge LLM evaluates answers
- Detects positional & verbosity bias

## How to Run
cd evaluator
python judge.py

## Output
results/output.txt

## Conclusion
LLMs can show bias and are not always reliable judges.
